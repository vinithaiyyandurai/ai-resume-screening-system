# AI Resume Screening & Job Recommendation System

This project is an Artificial Intelligence and Machine Learning based system that automatically analyzes resumes and recommends suitable job roles.
# Project Objective
To build an AI-driven solution that uses NLP and Machine Learning to screen resumes and predict relevant job roles.

# Features
- Resume text preprocessing using NLP
- Skill extraction from resume data
- Candidate classification (Fresher / Experienced)
- Job role recommendation using ML models

# Technologies Used
- Python
- Machine Learning (Scikit-learn)
- Natural Language Processing (TF-IDF)
- Pandas, NumPy

# Career Relevance
This project is suitable for entry-level AI Engineer and Machine Learning Engineer roles.
